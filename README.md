- еҚҸи®®иҪ¬жҚўпјҡOpenAI Chat еҸҜиҪ¬еҸ‘еҲ° Anthropic Messages жҲ– OpenAI Responses
- иҜ·жұӮж—Ҙеҝ—пјҡи®°еҪ•еҚҸи®®гҖҒжЁЎеһӢгҖҒе»¶иҝҹгҖҒTokenгҖҒжҲҗжң¬

## еҝ«йҖҹејҖе§Ӣ

### Docker ComposeпјҲжҺЁиҚҗпјү

```bash
mkdir lens && cd lens
curl -fsSLO https://raw.githubusercontent.com/dyedd/lens/main/docker-compose.yml
curl -fsSLO https://raw.githubusercontent.com/dyedd/lens/main/.env.example
cp .env.example .env
```

зј–иҫ‘ `.env`пјҢж №жҚ®йңҖиҰҒдҝ®ж”№й…ҚзҪ®йЎ№гҖӮеҝ…йЎ»и®ҫзҪ® `LENS_AUTH_SECRET_KEY`гҖӮ

еҰӮйңҖдҝ®ж”№ж•°жҚ®зӣ®еҪ•пјҢеҸӘж”№ `volumes` е·Ұдҫ§зҡ„е®ҝдё»жңәи·Ҝеҫ„пјҢеҸідҫ§ `/app/data` дҝқжҢҒдёҚеҸҳпјҡ

```yaml
volumes:
  - ./data:/app/data
```

еҗҜеҠЁпјҡ

```bash
docker compose pull
docker compose up -d
```

и®ҝй—® `http://127.0.0.1:3000`пјҢй»ҳи®ӨиҙҰеҸ· `admin/admin`гҖӮйҰ–ж¬Ўзҷ»еҪ•еҗҺиҜ·з«ӢеҚідҝ®ж”№й»ҳи®Өз®ЎзҗҶе‘ҳеҜҶз ҒгҖӮ

### Docker Run

```bash
mkdir -p data

docker run -d --name lens \
  --env-file .env \
  -p 3000:3000 \
  -v "$(pwd)/data:/app/data" \
  ghcr.io/dyedd/lens:latest
```

### жң¬ең°жһ„е»әй•ңеғҸ

```bash
docker compose -f docker-compose.yml -f docker-compose.local.yml up -d --build
```

`docker-compose.local.yml` йңҖиҰҒе’Ң `docker-compose.yml` ж”ҫеңЁеҗҢдёҖзӣ®еҪ•гҖӮд»“еә“дёӯе·ІжҸҗдҫӣиҜҘж–Үд»¶пјҢдјҡжҠҠй•ңеғҸеҗҚж”№жҲҗ `lens:local` е№¶д»ҺеҪ“еүҚжәҗз Ғжһ„е»әгҖӮ

еҰӮжһңеңЁзӢ¬з«ӢйғЁзҪІзӣ®еҪ•дёӯжң¬ең°жһ„е»әпјҢжүӢеҠЁеҲӣе»ә `docker-compose.local.yml`пјҡ

```yaml
services:
  app:
    image: lens:local
    build:
      context: .
      dockerfile: Dockerfile
```

жҠҠйЎ№зӣ®жәҗз Ғж”ҫеңЁеҗҢдёҖзӣ®еҪ•пјҢз„¶еҗҺжү§иЎҢпјҡ

```bash
docker compose -f docker-compose.yml -f docker-compose.local.yml up -d --build
```

д№ҹеҸҜд»ҘжүӢеҠЁжһ„е»әеҗҺзӣҙжҺҘиҝҗиЎҢпјҡ

```bash
docker build -t lens:local .

mkdir -p data

docker run -d --name lens \
  --env-file .env \
  -p 3000:3000 \
  -v "$(pwd)/data:/app/data" \
  lens:local
```

## дҪҝз”ЁжөҒзЁӢ

### 1. ж·»еҠ дёҠжёёз«ҷзӮ№

иҝӣе…Ҙ `/channels`пјҢж–°е»әз«ҷзӮ№пјҢеЎ«еҶҷ Base URL е’Ң API KeyпјҢеҸ‘зҺ°жҲ–жүӢеҠЁж·»еҠ жЁЎеһӢгҖӮ

еёёи§Ғ Base URLпјҡ

| дёҠжёёзұ»еһӢ            | Base URL зӨәдҫӢ                         | еҚҸи®®йҖүжӢ©         |
| ------------------- | ------------------------------------- | ---------------- |
| OpenAI              | `https://api.openai.com`              | OpenAI Chat / Responses / Embeddings |
| Anthropic           | `https://api.anthropic.com`           | Anthropic        |
| Gemini              | `https://generativelanguage.googleapis.com` | Gemini |
| NewAPI / Rerank     | `https://newapi.example.com`          | RerankпјҲйҖҸдј еҲ° `POST /v1/rerank`пјү |

### 2. еҲӣе»әжЁЎеһӢз»„

иҝӣе…Ҙ `/groups`пјҢж–°е»әжЁЎеһӢз»„пјҢйҖүжӢ©еҚҸи®®пјҢж·»еҠ дёҠжёёжЁЎеһӢпјҢйҖүжӢ©и·Ҝз”ұзӯ–з•Ҙпјҡ

- **иҪ®иҜў**пјҡеңЁжЁЎеһӢз»„жҲҗе‘ҳеұ•ејҖеҗҺзҡ„ key зә§еҖҷйҖүд№Ӣй—ҙе№іж»‘иҪ®иҜў
- **ж•…йҡңеҲҮжҚў**пјҡдјҳе…ҲдҪҝз”ЁеүҚйқўзҡ„жҲҗе‘ҳпјҢеӨұиҙҘеҗҺеҲҮеҲ°дёӢдёҖдёӘ key жҲ–жё йҒ“

**еҚҸи®®иҪ¬жҚў**пјҡеҪ“еүҚж”ҜжҢҒжҠҠ OpenAI Chat дёҠжёёеҠ е…Ҙ Anthropic жҲ– OpenAI Responses жЁЎеһӢз»„пјҢиҝҗиЎҢж—¶дјҡиҮӘеҠЁиҪ¬жҚўгҖӮ

### 3. еҸ‘ж”ҫзҪ‘е…і Key

иҝӣе…Ҙ `/api-keys`пјҢж–°е»ә KeyпјҢеӨҚеҲ¶ `sk-lens-...` з»ҷе®ўжҲ·з«ҜгҖӮ

### 4. е®ўжҲ·з«Ҝи°ғз”Ё

е®ўжҲ·з«ҜеҸӘйңҖиҰҒпјҡLens Base URL + зҪ‘е…і API Key + жЁЎеһӢз»„еҗҚз§°гҖӮ

| еүҚз«Ҝ   | Next.js 16гҖҒReact 19гҖҒTypeScriptгҖҒTanStack QueryгҖҒshadcn/ui    |

### PostgreSQL й…ҚзҪ®

PostgreSQL иҝһжҺҘдёІж јејҸпјҡ

```
postgresql+psycopg://з”ЁжҲ·еҗҚ:еҜҶз Ғ@дё»жңә:з«ҜеҸЈ/ж•°жҚ®еә“еҗҚ
```

зӨәдҫӢпјҡ

```bash
LENS_DATABASE_URL=postgresql+psycopg://lens:password@postgres.example.com:5432/lens
```

**1Panel зӯүе®№еҷЁеҢ–зҺҜеўғй…ҚзҪ®жҠҖе·§**пјҡ

еҰӮжһң Lens е’Ң PostgreSQL йғЁзҪІеңЁеҗҢдёҖеҸ°жңҚеҠЎеҷЁпјҢжҺЁиҚҗжҠҠдёӨдёӘе®№еҷЁж”ҫеҲ°еҗҢдёҖдёӘ Docker зҪ‘з»ңпјҲдҫӢеҰӮ 1Panel зҡ„ `1panel-network`пјүпјҢз„¶еҗҺз”Ё PostgreSQL е®№еҷЁеҗҚдҪңдёәдё»жңәеҗҚпјҡ

```bash
LENS_DATABASE_URL=postgresql+psycopg://lens:password@postgresql:5432/lens
```

иҝҷйҮҢз¬¬дёҖдёӘ `lens` жҳҜж•°жҚ®еә“з”ЁжҲ·еҗҚпјҢжңҖеҗҺдёҖдёӘ `lens` жҳҜж•°жҚ®еә“еҗҚпјӣ`postgresql` жҳҜ PostgreSQL е®№еҷЁеҗҚпјҢйңҖиҰҒжҢүе®һйҷ…е®№еҷЁеҗҚи°ғж•ҙгҖӮ

**SQLite йҖӮеҗҲжң¬ең°жөӢиҜ•е’ҢиҪ»йҮҸйғЁзҪІпјҢз”ҹдә§зҺҜеўғжҲ–й«ҳе№¶еҸ‘еңәжҷҜе»әи®®дҪҝз”Ё PostgreSQLгҖӮ**

## ж•°жҚ®еә“иҝҒз§»

```bash
lens db upgrade                               # еҚҮзә§еҲ°жңҖж–°
lens db downgrade                             # еӣһйҖҖдёҖжӯҘ
lens db revision -m "describe your change"    # з”ҹжҲҗж–°иҝҒз§»
```

д»Һ SQLite еҲҮжҚўеҲ° PostgreSQLпјҡеңЁ `/backups` еҜјеҮәй…ҚзҪ® вҶ’ дҝ®ж”№ `LENS_DATABASE_URL` вҶ’ еҗҜеҠЁ Lens вҶ’ еҜје…Ҙй…ҚзҪ®гҖӮ
